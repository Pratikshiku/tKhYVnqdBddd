### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

- `Visual Studio Code(VScode)`
- `Navicat`

### 三、功能描述
系统分为（身份）：普通用户、管理员。

**普通用户功能：**
1. 登录
2. 修改密码
3. 个人信息
4. 用户管理
5. 车辆管理
6. 保养登记管理
7. 事故登记管理
8. 维修登记管理
9. 违章登记管理


**管理员功能：**
1. 登录
2. 个人中心(修改密码、个人信息）
3. 基础数据管理（保养类型管理、车辆类型管理、事故类型管理、维修类型管理、违章类型管理、准驾车型管理）
4. 员工管理
5. 用户管理
6. 车辆管理
7. 保养登记管理
8. 事故登记管理
9. 维修登记管理
10. 违章登记管理
